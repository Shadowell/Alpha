# 背景 / 目标

用户要求在 Alpha 页面中新增一个独立的“图形选股”模块，把 `FirstLimit Alpha` 模型接入页面展示，并参考现有漏斗设计提供三池结构。

本次目标：

- 在前端新增独立 Tab/页面模块，用于展示 `FirstLimit Alpha` 的模型选股结果
- 后端提供可触发、可持久化的图形选股快照接口
- 使用现有训练好的 `FirstLimit Alpha` baseline 模型进行打分
- 形成候选池 / 重点池 / 买入池三池结构
- 卡片支持点击后查看 K 线 / Kronos 预测，保持与现有页面体验一致

## 验收标准

- 页面左侧导航出现新的图形选股入口
- 图形选股页面能展示 3 个池子，并显示模型评分与关键信号
- 点击卡片可打开预测/K线查看
- 后端可通过 API 触发运行并返回快照
- 最新快照会持久化，不刷新页面仍可读取
- `README.md` 已同步说明该模块

## 风险与回滚

- 风险：训练样本构建依赖未来标签，无法直接用于最新交易日推理
- 应对：新增“无标签候选样本”构建路径，专门服务在线推理，不复用训练标签逻辑
- 风险：模型分数分布与百分比型预测模块不同，固定阈值可能过稀或过密
- 应对：提供独立阈值配置，并以当前样本分布给出默认值
- 回滚：本次改动以新增页面和 API 为主，不影响原有 funnel / notice / predict 页面

## 关键决策记录

- 图形选股作为独立 Tab，而不是合并进“预测选股”
- 第一版在线页面默认接入 `FirstLimit Alpha baseline`，时序模型结果保留为离线对比，不直接替代线上主模型
- 卡片详情复用现有 `openPredictModal`，避免重复构建 K 线交互
