## 风险与回滚

- 风险：现有本地数据只有日线，缺失盘口/分时级微结构特征
- 应对：将微结构特征设计为可选层，baseline 默认只依赖稳定日线与概念数据
- 风险：全市场训练耗时较长，首轮验证不适合在请求内跑完整全量回测
- 应对：代码支持全量运行，但先以可复现抽样/有限样本验证接口与流程
- 回滚：新增内容以独立模块和 API 增量接入，不修改用户现有 `daban` 相关未提交改动；如需回退，可仅移除新增文件和路由挂载

## 关键决策记录

- 训练默认优先使用本地已可用的 `lightgbm`，保留 `sklearn` 退化路径，避免环境依赖阻塞
- 数据源优先复用现有 `KlineSQLiteStore` 与 `AkshareDataProvider`
- FastAPI 接口采用独立 router 注入，避免继续扩大 `app/main.py` 的职责
- 第一版预测时点统一为“首板收盘后，次日竞价/开盘介入”，以保证可复现和避免盘中未来函数
